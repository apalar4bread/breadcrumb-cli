# breadcrumb-cli

> A CLI tool that tracks and replays your terminal session steps as reproducible shell scripts.

---

## Installation

```bash
pip install breadcrumb-cli
```

Or install from source:

```bash
git clone https://github.com/yourname/breadcrumb-cli.git && cd breadcrumb-cli && pip install .
```

---

## Usage

Start tracking your terminal session:

```bash
breadcrumb start
```

Run your commands as usual. When you're done, stop tracking and export a reproducible script:

```bash
breadcrumb stop
breadcrumb export --output setup.sh
```

Replay a previously recorded session:

```bash
breadcrumb replay setup.sh
```

List recorded sessions:

```bash
breadcrumb list
```

Example output after export:

```bash
#!/bin/bash
# Generated by breadcrumb-cli
mkdir my-project
cd my-project
python -m venv .venv
source .venv/bin/activate
pip install requests
```
